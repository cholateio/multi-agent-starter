# Kit v4.4 Project Manifest（PROJECT.toml + proj）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 每個專案一份機器可讀的 `PROJECT.toml`（狀態／起始指令／付費服務），由 kit 發佈骨架，`bin/proj` 做跨專案彙總。

**Architecture:** kit 是發佈管道——install mode no-clobber 鋪骨架、update mode deploy-if-absent 補鋪、維護慣例放 kit-owned rule 隨 `--update` 推平。彙總工具是零依賴 Python 單檔，掃 `$PROJ_ROOT`（預設 `$HOME`）第一層子目錄。

**Tech Stack:** bash（init.sh、smoke tests，比照現有：無 jq、無第三方工具）；Python 3.11+ stdlib only（`tomllib`）。

**Spec:** `docs/superpowers/specs/2026-07-11-kit-v4.4-project-manifest-design.md`（權威需求來源）。

## Global Constraints

- Python 3.11+，只用 stdlib（`tomllib` / `subprocess` / `pathlib` / `unicodedata`）；本機為 3.12.3。
- 測試沿用 `tests/smoke.sh` 的隔離手法：mktemp fixture、絕不碰 kit repo working tree、PASS/FAIL 計數、失敗才 dump。
- `.claude/rules/` 總量上限 **20480 bytes**（smoke.sh 的 budget 測項把關；現用 17869B，新 rule 檔必須 ≤ ~2KB）。
- `PROJECT.toml` 是 **user-owned**：install no-clobber、update deploy-if-absent、永不 diff 警告、永不覆蓋。
- `status` 封閉集合：`active | mvp | done | paused | abandoned`。過時閾值：最後 commit 比 `updated` 晚 **30 天**以上。
- commit 訊息比照 repo 慣例（`feat:` / `docs:` / `test:` 前綴，一行主旨）。

**模型建議（kit-workflow 要求的 per-phase proposal）：** Task 1–7 由主對話（Fable 5 / medium effort）inline 執行——檔案位置全部已知、單檔改動小，派工不經濟；Task 8 的 11 專案掃描屬「必須派工」情境，用 sonnet 檔位 subagent 平行掃描、主對話驗收。升級觸發：smoke harness 連卡 2 次帶診斷升 high effort。

```bash
#!/usr/bin/env bash
# tests/proj-smoke.sh - acceptance tests for bin/proj (kit v4.4, spec §6-7)
#
# 隔離手法比照 smoke.sh:mktemp fixture、絕不碰 kit repo、PASS/FAIL 計數。
set -uo pipefail

KIT_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
PROJ="$KIT_ROOT/bin/proj"

WORK="$(mktemp -d "${TMPDIR:-/tmp}/proj-smoke.XXXXXX")"
cleanup() { rm -rf "$WORK"; }
trap cleanup EXIT
trap 'exit 130' INT
trap 'exit 143' TERM

PASS_COUNT=0; FAIL_COUNT=0
pass() { PASS_COUNT=$((PASS_COUNT+1)); printf 'PASS %s\n' "$1"; }
fail() { FAIL_COUNT=$((FAIL_COUNT+1)); printf 'FAIL %s (%s)\n' "$1" "$2"; }
assert_eq() { if [ "$2" = "$3" ]; then pass "$1"; else fail "$1" "got [$2] want [$3]"; fi }
assert_contains() { if printf '%s\n' "$2" | grep -qF -- "$3"; then pass "$1"; else fail "$1" "missing text: [$3]"; fi }
assert_not_contains() { if printf '%s\n' "$2" | grep -qF -- "$3"; then fail "$1" "unexpectedly contains: [$3]"; else pass "$1"; fi }

if [ ! -x "$PROJ" ]; then
  echo "FAIL setup (bin/proj not found/executable at $PROJ)"
  echo "passed 0, failed 1"; exit 1
fi

# --- fixtures: 正常 manifest / 無 manifest 純 git / 壞 TOML / 隱藏目錄 ---
ROOT="$WORK/root"
mkdir -p "$ROOT/good-proj" "$ROOT/bare-proj/.git" "$ROOT/broken-proj" "$ROOT/.hidden"

cat > "$ROOT/good-proj/PROJECT.toml" <<'EOF'
name = "good-proj"
status = "mvp"
status_note = "單影片可用"
updated = 2020-01-01

[commands]
summary = "uv run good-proj <url>"

[[paid]]
service = "OpenAI API"
billing = "按用量"
monthly_est = "~$3"
cancel = "拿掉 .env 的 key"

[[paid]]
service = "Supabase"
billing = "free-tier"
monthly_est = "$0"
EOF

printf 'name = "broken\n' > "$ROOT/broken-proj/PROJECT.toml"

# run: $@ = proj args; sets OUT / ERR / CODE
run() {
  OUT="$(env PROJ_ROOT="$ROOT" "$PROJ" "$@" 2>"$WORK/stderr")"; CODE=$?
  ERR="$(cat "$WORK/stderr")"
}

# --- proj (list) ---
run
assert_eq "list: exit 0" "$CODE" "0"
assert_contains "list: good-proj status shown" "$OUT" "mvp"
assert_contains "list: non-free service \$-marked" "$OUT" "\$OpenAI API"
assert_not_contains "list: free-tier not \$-marked" "$OUT" "\$Supabase"
assert_contains "list: bare-proj is 未登記" "$OUT" "未登記"
assert_contains "list: broken manifest flagged in table" "$OUT" "manifest 損壞"
assert_contains "list: parse warning on stderr" "$ERR" "解析失敗"
assert_not_contains "list: hidden dir excluded" "$OUT" ".hidden"

# --- stale 偵測: good-proj 給一個今天的 commit,updated=2020 → ⚠ ---
GIT_ID="$WORK/gitcfg"
printf '[user]\n\tname = T\n\temail = t@t.test\n' > "$GIT_ID"
gitq() { GIT_CONFIG_NOSYSTEM=1 GIT_CONFIG_GLOBAL="$GIT_ID" git -C "$ROOT/good-proj" "$@" >/dev/null 2>&1; }
if gitq init -q -b main && gitq add -A && gitq commit -q -m x; then
  run
  assert_contains "list: stale manifest marked" "$OUT" "⚠"
else
  fail "list: stale manifest marked" "git fixture setup failed"
fi

# --- proj <name> ---
run good-proj
assert_eq "detail: exit 0" "$CODE" "0"
assert_contains "detail: command copyable verbatim" "$OUT" "uv run good-proj <url>"
assert_contains "detail: cancel instruction shown" "$OUT" "拿掉 .env 的 key"

run no-such-proj
assert_eq "detail: unknown project exits 1" "$CODE" "1"

echo
echo "passed $PASS_COUNT, failed $FAIL_COUNT"
[ "$FAIL_COUNT" -eq 0 ] || exit 1
exit 0
```

```python
#!/usr/bin/env python3
"""proj — 跨專案 PROJECT.toml 彙總(kit v4.4)。

用法:
  proj              全專案總覽表格(狀態/說明/付費服務/最後 commit)
  proj <name>       單一專案完整資訊(起始指令可直接複製)
  proj money        只列有付費服務的專案,每服務一列
  proj remote       gh repo list 對照本機,列出未 clone 的 repo

掃描根目錄: $PROJ_ROOT(預設 $HOME)第一層子目錄中,有 .git 或
PROJECT.toml 者視為專案。Python 3.11+,零第三方依賴。
"""
import datetime
import os
import shutil
import subprocess
import sys
import tomllib
import unicodedata
from pathlib import Path

VALID_STATUSES = {"active", "mvp", "done", "paused", "abandoned"}
STALE_DAYS = 30


def warn(msg):
    print(f"proj: {msg}", file=sys.stderr)


def find_projects(root):
    try:
        entries = sorted(root.iterdir())
    except OSError as e:
        warn(f"無法讀取 {root}: {e}")
        return []
    return [d for d in entries
            if d.is_dir() and not d.name.startswith(".")
            and ((d / ".git").exists() or (d / "PROJECT.toml").is_file())]


def load_manifest(d):
    """回傳 (data, err): 無檔 (None, None);壞檔 (None, str);正常 (dict, None)。"""
    p = d / "PROJECT.toml"
    if not p.is_file():
        return None, None
    try:
        with open(p, "rb") as f:
            data = tomllib.load(f)
    except (tomllib.TOMLDecodeError, OSError) as e:
        return None, str(e)
    status = data.get("status")
    if status is not None and status not in VALID_STATUSES:
        warn(f'{d.name}: status "{status}" 不在 {sorted(VALID_STATUSES)},照列')
    return data, None


def last_commit_date(d):
    if not (d / ".git").exists():
        return None
    try:
        r = subprocess.run(["git", "-C", str(d), "log", "-1", "--format=%cs"],
                           capture_output=True, text=True, timeout=10)
    except (OSError, subprocess.TimeoutExpired):
        return None
    try:
        return datetime.date.fromisoformat(r.stdout.strip())
    except ValueError:
        return None


def is_stale(data, commit_date):
    upd = data.get("updated")
    if isinstance(upd, datetime.datetime):
        upd = upd.date()
    if not isinstance(upd, datetime.date) or commit_date is None:
        return False
    return (commit_date - upd).days > STALE_DAYS


# --- 等寬表格(CJK 全形字算 2 格,不然中文欄位會歪) ---

def disp_width(s):
    return sum(2 if unicodedata.east_asian_width(c) in ("W", "F") else 1
               for c in s)


def pad(s, width):
    return s + " " * (width - disp_width(s))


def print_table(headers, rows):
    widths = [disp_width(h) for h in headers]
    for r in rows:
        for i, cell in enumerate(r):
            widths[i] = max(widths[i], disp_width(cell))
    emit = lambda cells: print(
        "  ".join(pad(cells[i], widths[i]) for i in range(len(cells))).rstrip())
    emit(headers)
    emit(["-" * w for w in widths])
    for r in rows:
        emit(r)


def paid_summary(data):
    parts = []
    for p in data.get("paid", []):
        name = str(p.get("service", "?"))
        if str(p.get("billing", "")) != "free-tier":
            name = "$" + name
        parts.append(name)
    return ", ".join(parts)


def cmd_list(root):
    rows = []
    for d in find_projects(root):
        data, err = load_manifest(d)
        cdate = last_commit_date(d)
        cstr = cdate.isoformat() if cdate else "-"
        if err:
            warn(f"{d.name}/PROJECT.toml 解析失敗: {err}")
            rows.append([d.name, "manifest 損壞", "", "", cstr])
        elif data is None:
            rows.append([d.name, "未登記", "", "", cstr])
        else:
            status = str(data.get("status", "?"))
            if is_stale(data, cdate):
                status += " ⚠"
            rows.append([d.name, status, str(data.get("status_note", "")),
                         paid_summary(data), cstr])
    print_table(["專案", "狀態", "說明", "付費服務", "最後 commit"], rows)
    return 0


def cmd_detail(root, name):
    d = root / name
    if not d.is_dir():
        warn(f"找不到專案目錄 {d}")
        return 1
    data, err = load_manifest(d)
    if err:
        warn(f"{name}/PROJECT.toml 解析失敗: {err}")
        return 1
    if data is None:
        warn(f"{name} 沒有 PROJECT.toml(未登記)")
        return 1
    cdate = last_commit_date(d)
    stale = "  ⚠ manifest 可能過時" if is_stale(data, cdate) else ""
    print(f"{data.get('name', name)}  [{data.get('status', '?')}]{stale}")
    if data.get("status_note"):
        print(data["status_note"])
    print(f"updated: {data.get('updated', '?')}   "
          f"最後 commit: {cdate.isoformat() if cdate else '-'}")
    if data.get("commands"):
        print("\n指令:")
        for k, v in data["commands"].items():
            print(f"  {k}:")
            print(f"    {v}")
    paid = data.get("paid", [])
    if paid:
        print("\n付費服務:")
        for p in paid:
            print(f"  - {p.get('service', '?')} | {p.get('billing', '?')}"
                  f" | {p.get('monthly_est', '?')}")
            if p.get("cancel"):
```

```python
def cmd_money(root):
    rows = []
    for d in find_projects(root):
        data, _ = load_manifest(d)
        if not data:
            continue
        for p in data.get("paid", []):
            rows.append([d.name, str(p.get("service", "?")),
                         str(p.get("billing", "?")),
                         str(p.get("monthly_est", "?")),
                         str(p.get("cancel", ""))])
    if not rows:
        print("沒有登記任何付費服務。")
        return 0
    print_table(["專案", "服務", "計費", "月費估計", "取消方式"], rows)
    return 0


def cmd_remote(root):
    if shutil.which("gh") is None:
        warn("需要 gh CLI(安裝後 gh auth login)")
        return 1
    r = subprocess.run(
        ["gh", "repo", "list", "--json", "name,updatedAt", "--limit", "200"],
        capture_output=True, text=True)
    if r.returncode != 0:
        warn(f"gh repo list 失敗: {r.stderr.strip()}")
        return 1
    import json
    repos = json.loads(r.stdout)
    local = {d.name for d in find_projects(root)}
    rows = [[x["name"], x.get("updatedAt", "")[:10]]
            for x in sorted(repos, key=lambda x: x["name"])
            if x["name"] not in local]
    if not rows:
        print("GitHub 上所有 repo 都已 clone。")
        return 0
    print_table(["未 clone 的 repo", "最後更新"], rows)
    return 0
```
